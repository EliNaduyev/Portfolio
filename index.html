Projects

XSS Scanner

Following OWASP TOP 10 (the top ten most critical web application security risk) I decided to build an XSS Scanner. In the Frontend I used Vanilla Javascript and in the Backend I used NodeJS and Express for the server and Puppeteer for automation.

Puppeteer is an open source library developed by Google that was built in purpose of automating and simplifying tests and development. The scanner getting link from the user and scan the website for XSS vulnerability by injecting malicious script and check if the website is running the script. The scanner is trying to inject variety of scripts for better results.
